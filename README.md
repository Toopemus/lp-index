# lp-index

App for cataloging my personal [lp](https://en.wikipedia.org/wiki/LP_record) collection.

## Stack

Express.js + TypeScript, Sequelize ORM

## Starting the app

```shell
# Install packages
npm install

# Dev database daemon
docker compose up -d

# Compile TypeScript files (you only need to do this on the first time)
npx tsc

# Dev server
npm run dev
```
